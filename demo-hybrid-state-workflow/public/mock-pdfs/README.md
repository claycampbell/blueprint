# Mock Correction Letter PDFs

This directory contains mock PDF correction letters for the entitlement subprocess demo.

These are HTML-based mock PDFs that can be viewed in an iframe to simulate the PDF viewer experience.

## Files

- `2024-5678-round2.html` - Capitol Hill Avenue, Round 2 (8 items)
- `2024-9012-round1.html` - University Way NE, Round 1 (6 items)
- `2024-3456-round3.html` - Greenwood Avenue North, Round 3 (4 items)

## Usage

These mock PDFs match the correction items defined in `src/data/entitlementProperties.ts` and can be displayed in the CorrectionTriageView PDF viewer.
